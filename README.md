# Netflix EDA Project 🎬

## About
Exploratory Data Analysis on Netflix Movies and TV Shows dataset
from Kaggle as part of PlutoAcademy AI & ML Internship Program.

## Dataset
Netflix Movies and TV Shows — Kaggle

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Steps Completed
1. Load & Inspect Data
2. Clean the Data
3. Exploratory Data Analysis (5 Questions)
4. Visualizations (6 Charts)
5. Insights Report

## Key Insights
1. Movies dominate Netflix at 70%
2. USA leads content production
3. Netflix grew rapidly between 2016-2019
4. Most content is released after 2000
